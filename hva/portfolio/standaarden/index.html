Hallo en welkom op mijn website!

Zoals men ongetwijfeld kan zien heb ik absoluut geen voorliefde voor vormgeving. Sterker nog, ik verafschuw het. Gelukkig ben ik wel in staat om mooie features en een digitale versie van "pimp my ride" toe te passen. De site maakt gebruik van brute HTML5 tags en stoere CSS3. Man man, dat is toch prachtig? Verder ben ik ook zo briljant geweest om een geniale JavaScript functie toe te voegen in de knop hieronder. Test het! Klik erop!

All gekkigheid terzijde, deze site bevat alle technische functies die zijn behandeld. En daarnaast al mijn teksten van Tekst in Context

Daarnaast om te laten zien dat ik ook nog afbeeldingen kan importeren, een plaatje van een geit

Bronvermelding:

  • Subtle Patterns voor enkele patronen
  • StackOverflow voor hulp met het menu
  • Code Academy voor mijn basis javascript kennis
  • Wikipedia voor de geit

En tot slot, een plaatje van een vliegende nyan kat!
